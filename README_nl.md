## HackCube-Special Wat is het?

### HackCube-Special is een draagbaar radioplatform voor hardwaredetectie om op meerdere radiofrequentiebanden reverse-analyse van omringende RF-signalen te vergemakkelijken.

## Wat kan het doen.
